# Hope of South lebanon

A humanitarian initiative connecting blood donors with those in need across Lebanon.

# Technologies Used

- HTML
- CSS
- JAVASCRIPT
- FIREBASE FIRESTORE

# Features

- Register as a blood donor
- Search for fonors by blood type and region
- Blood type compatibility checker
- Urgent blood request form
- Fully responsive design

# Note

This project uses Firebase Firestore
The firebase.js config file is not included for security reasons
